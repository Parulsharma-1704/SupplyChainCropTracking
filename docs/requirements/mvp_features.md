# MVP FEATURES - SUPPLY CHAIN & CROP TRACKING

## CORE USER STORIES

### FARMER USER
1. As a farmer, I want to register my farm details
2. As a farmer, I want to add my crops with planting/harvest dates
3. As a farmer, I want to generate a QR code for each crop batch
4. As a farmer, I want to see price predictions for my crops
5. As a farmer, I want to update shipment status when I send crops
6. As a farmer, I want to view my crop history and earnings

### DISTRIBUTOR USER
7. As a distributor, I want to browse available crops from farmers
8. As a distributor, I want to filter crops by type, quality, location
9. As a distributor, I want to purchase crops from farmers
10. As a distributor, I want to track purchased shipments
11. As a distributor, I want to verify crop quality certifications

### ADMIN USER
12. As an admin, I want to manage user accounts
13. As an admin, I want to view system analytics
14. As an admin, I want to add/manage market price data for ML

### SYSTEM REQUIREMENTS
15. Price prediction based on crop type, season, quality
16. Real-time shipment tracking with status updates
17. QR code generation and scanning for verification
18. User authentication and authorization
19. Dashboard with analytics and charts

## MVP EXCLUSIONS (Phase 2)
- Mobile app (web only)
- Advanced IoT sensor integration
- Complex blockchain implementation
- Multi-language support
- Advanced reporting
- Bulk operations
